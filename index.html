📅 Resumo de Códigos Básicos: Guia Rápido de HTML

Para ajudar todo mundo a colocar a mão na massa no laboratório, aqui está o nosso guia de sobrevivência HTML. Use essas tags para estruturar o seu site!

  • <h1> até <h6>: 🏷️ Títulos e subtítulos. O <h1> é o principal e maior da página.
  • <p>: ✍️ Parágrafo de texto comum.
  • <strong>: ⚠️ Deixa o texto em negrito para dar destaque.
  • <em>: ✨ Deixa o texto em itálico para dar ênfase.
  • <ul>: 📋 Cria uma lista com marcadores (bolinhas).
  • <ol>: 🔢 Cria uma lista numerada automática (1, 2, 3...).
  • <li>: 🔹 Cada item individual de uma lista (deve ficar dentro de ul ou ol).
  • <br>: ↩️ Quebra de linha simples (não precisa de tag de fechamento).

💡 Dica de Web Design: Como centralizar imagens com CSS

Muitas vezes, quando usamos a propriedade text-align: center; no CSS, ela funciona muito bem para textos, mas não mexe na nossa imagem. Para resolver esse problema de forma isolada e deixar o nosso mascote perfeitamente no meio da página, aprendemos um truque muito importante!


img {
  width: 400px;
  height: 400px;
  display: block; /* Transforma a imagem em bloco */
  margin: 0 auto; /* Alinha os lados automaticamente */
}

🛠️ Entendendo as propriedades:

  • display: block;: Por padrão, as imagens ficam na mesma linha dos textos. Ao mudar para bloco, dizemos ao navegador que a imagem deve ocupar a sua própria linha exclusiva no site.
  • margin: 0 auto;: O valor auto indica para o navegador calcular o espaço restante nas laterais esquerda e direita de forma igual, empurrando o elemento exatamente para o centro da tela.

Agora que dominamos esse truque, nossos layouts vão ficar muito mais organizados e profissionais! Testem nos seus projetos individuais e vejam a mágica acontecer.

📅 02 de Julho de 2026 — Aprendendo a Organizar o Blog com Linhas Divisórias

Com tantas postagens legais surgindo no nosso site, é muito importante manter o design limpo e fácil de ler. Hoje aprendemos a usar a tag <hr> (Horizontal Rule), que serve para criar linhas horizontais e separar os conteúdos de forma profissional!

🛠️ Como colocar a linha no seu código HTML?

Diferente de tags como <p> ou <h2>, a tag <hr> é uma tag de **fechamento automático**. Isso significa que você não precisa colocar um </hr> no final. Basta digitar a tag exatamente onde quer a divisão!

<p>Texto do primeiro post do blog...</p>


<hr style="border: 0; border-top: 1px solid #bd346b; margin: 30px auto; max-width: 90%; opacity: 0.5;">

<p>Texto do segundo post do blog...</p>

🎨 Entendendo os estilos usados na linha:

  • border-top: 1px solid #bd346b;: Define a espessura fina (1px) e a cor da linha usando o tom oficial do nosso blog.
  • max-width: 90%;: Faz com que a linha não encoste totalmente nas laterais da página, deixando o visual mais elegante.
  • margin: 30px auto;: Garante um espaço livre de 30 pixels acima e abaixo da linha e a mantém perfeitamente centralizada.
  • opacity: 0.5;: Deixa a linha suave e levemente transparente para não brigar com a leitura do texto.

💡 Desafio para a aula de hoje: Abra seu arquivo no VS Code e aplique essa linha estilizada entre as três postagens que vocês criaram nas semanas anteriores!

📅 02 de Julho de 2026 — Como Criar um Carrossel de Vídeos no Blog

Se o seu grupo tem mais de um vídeo ou quer organizar indicações de clipes e podcasts de forma moderna, o Carrossel Horizontal é a melhor solução! Com ele, os vídeos ficam lado a lado e o usuário pode navegar deslizando para os lados, igualzinho aos aplicativos de streaming.

🛠️ Código Prático para o seu HTML

Para fazer a mágica acontecer, criamos uma caixa organizadora (uma tag <div>) com propriedades CSS que ativam a rolagem horizontal automática. Copie o modelo abaixo e substitua pelos links dos seus vídeos:

<!-- Início do Carrossel -->
<div style="display: flex; overflow-x: auto; gap: 20px; padding: 15px; scroll-snap-type: x mandatory; -webkit-overflow-scrolling: touch;">

  <!-- Vídeo 1 -->
  <iframe width="420" height="236" src="https://youtube.com" frameborder="0" allowfullscreen style="flex: 0 0 auto; scroll-snap-align: start; border-radius: 8px;"></iframe>

  <!-- Vídeo 2 -->
  <iframe width="420" height="236" src="https://youtube.com" frameborder="0" allowfullscreen style="flex: 0 0 auto; scroll-snap-align: start; border-radius: 8px;"></iframe>

  <!-- Vídeo 3 -->
  <iframe width="420" height="236" src="https://youtube.com" frameborder="0" allowfullscreen style="flex: 0 0 auto; scroll-snap-align: start; border-radius: 8px;"></iframe>

</div>
<!-- Fim do Carrossel -->

🧠 Como essa mágica do CSS funciona?

  • display: flex;: Coloca todos os vídeos um ao lado do outro em uma única linha.
  • overflow-x: auto;: Cria a barra de rolagem horizontal sempre que os vídeos passarem do limite da tela.
  • gap: 20px;: Adiciona um espaçamento de 20 pixels entre os vídeos para que eles não fiquem colados.
  • flex: 0 0 auto;: Propriedade fundamental dentro dos <iframe> que impede o navegador de esmagar o tamanho dos vídeos.

💡 Dica de ouro: Lembre-se de usar a URL de incorporação do YouTube (com o termo /embed/ no link) para que os vídeos funcionem diretamente na sua página!
